# Vercel Login Fix - Quick Reference Card

## 🚨 You're Getting This Error:
```
Server returned non-JSON: 500
FUNCTION_INVOCATION_FAILED
```

## ✅ FIX IT IN 5 MINUTES:

### 1️⃣ Add These Environment Variables to Vercel

Go to: **vercel.com → Your Project → Settings → Environment Variables**

Add 4 variables:
```
MONGODB_URI = [Your MongoDB Connection String]
JWT_SECRET = [Any random string, 32+ characters]
GOOGLE_CLIENT_ID = [Your Google OAuth ID]
GOOGLE_CLIENT_SECRET = [Your Google OAuth Secret]
```

### 2️⃣ Allow Vercel to Access MongoDB

Go to: **MongoDB Atlas → Network Access**
- Click "Add IP Address"
- Select "Allow Access From Anywhere"
- Click "Confirm"

### 3️⃣ Push Code & Redeploy

```bash
git add .
git commit -m "Fix login endpoint"
git push
```

## ⏱️ That's it! Should take 5 minutes.

---

## 🔧 If Still Not Working:

### Check These (In Order):

1. **All 4 env vars set?**
   - Vercel → Settings → Environment Variables
   - Verify all 4 have values

2. **MongoDB IP whitelisted?**
   - MongoDB Atlas → Network Access
   - Should show 0.0.0.0/0

3. **Check Vercel Logs**
   ```bash
   vercel logs your-project-name
   ```
   Look for: "Connected to MongoDB successfully"

4. **Test Locally**
   ```bash
   npm install
   npm run dev
   ```

5. **Run Diagnostic Script**
   ```bash
   ./debug-vercel-login.sh
   ```
